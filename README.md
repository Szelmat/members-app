# Tagkezelő alkalmazás

## Backend
### Követelmények
* Python 3
* pip

### Szükséges modulok telepítése
```cmd
cd backend
pip install pipenv
pipenv shell
pipenv install
```
### A szerver indítása
```cmd
python manage.py runserver
```
---
## Frontend
### Követelmények
* npm

### Szükséges modulok telepítése
```cmd
cd frontend
npm install
```
### Frontend indítása
```cmd
npm start
```
